## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/rishunandan1234/to_do_list.git
```

2. Navigate to the project directory:
```bash
cd to_do_list
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

5. Open your browser and visit:
```
http://localhost:5173
```
